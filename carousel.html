Como hacer un carousel de imagenes.

El carrusel es una presentación de diapositivas para el ciclismo a través de una serie de contenido, construido con CSS 3D y un poco de JavaScript. Funciona con una serie de imágenes, texto o marcado personalizado. También incluye soporte para controles e indicadores anteriores / próximos.

Crearemos un carousel como el siguiente


Previous Next

Paso 1.- Primeramente crear tu pagina html donde crearas el carousel
Paso 2.- Despues crear una carpeta en la raiz para almacenar las imagenes que iran en el carousel.
Paso 3.- Colocaremos el siguiente Codigo en nuestra pagina html

Paso 4.- En cada etiqueta de imagen en "src" coloca la url de la imagen deseada como ejemplo images/img1.jpg
Paso 5.- Ejecuta tu HTML y visualiza tu carousel

Recomendaciones:

Asegúrese de establecer una identificación única en los .carouselcontroles opcionales, especialmente si está utilizando múltiples carruseles en una sola página.

Código carousel

                              

                      <div id="carouselExampleSlidesOnly" class="carousel slide" data-ride="carousel">
                          <div class="carousel-inner" role="listbox">
                             <div class="carousel-item active">
                                  <img class="d-block img-fluid" src="..." alt="First slide">
                              </div>
                              <div class="carousel-item">
                                  <img class="d-block img-fluid" src="..." alt="Second slide">
                              </div>
                              <div class="carousel-item">
                                  <img class="d-block img-fluid" src="..." alt="Third slide">
                              </div>
                         </div>
                      </div>
